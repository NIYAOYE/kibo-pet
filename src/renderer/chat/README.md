# src/renderer/chat — 对话框 / 气泡 UI

用户与宠物交流的界面。点击宠物或按全局热键唤出;显示对话与台词气泡。

## 职责
- 唤出/收起对话框(点击宠物、全局热键触发)。
- 输入框:把用户文本发给主进程。
- 展示**流式回复**(逐 token)与**口癖台词气泡**。
- 呈现"正在思考/说话"等状态提示(与 [pet/](../pet/) 动画配合)。

## 交互
- → [ipc](../../main/ipc/):发送用户输入、唤出/收起请求。
- ← [ipc](../../main/ipc/):接收 [agent](../../main/agent/) 的回复流、[lines](../../main/lines/) 的台词文本。
- → [voice/](../voice/):有音频指令时触发播放。
