# src/main/shell — 窗口 / 热键 / 托盘(系统集成)

主进程侧的 OS 集成层。**只有主进程能创建 BrowserWindow、注册全局热键、放托盘图标**,所以这些都在这里。

## 职责
- 创建承载宠物的**透明、无边框、始终置顶**窗口(`BrowserWindow`,加载 `src/renderer`)。
- 注册**全局热键**(`globalShortcut`)唤出对话框。
- 系统**托盘**图标与菜单(显示/隐藏、设置、退出)。
- 窗口生命周期(单例、开机自启可选、多显示器定位)。

## 交互
- → `src/renderer`:装载渲染页面;经 `ipc/` 通知"热键被按下 / 请求唤出对话框"。
- ← `src/renderer`:接收"移动窗口到 (x,y) / 调整点击穿透区域"等请求(宠物拖拽、避让)。
- ← `config/`:读取热键绑定、置顶/自启等偏好。
