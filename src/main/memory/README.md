# src/main/memory — 分层记忆系统(标准档)

给宠物"记性"。分两层,纯本地存储,整个记忆目录**可被用户拷走**备份/迁移。

## 结构
- **短期记忆**:当前对话窗口(最近 N 轮)+ **滚动工作记忆摘要**(自动总结,防上下文爆炸)。
- **长期记忆**:
  - **结构化事实库**:用户画像/偏好/关键事实,存为人类可读文件(JSON/Markdown)。
  - **语义记忆库**:本地文件型向量库,按相关性(RAG)召回。
  - **写入策略**:判断什么值得记、什么可忽略。

## 交互
- ← [agent/](../agent/):每轮结束写入新记忆 / 更新工作记忆;每轮开始被召回。
- → [providers/](../providers/):调用 Embedding Provider 做向量化与召回。
- → [persona/](../persona/):把召回结果交给 system prompt 组装。
- → 用户数据目录:落盘于可移植的 `memory/` 目录(卸载不删)。
